example call

http://namjai.eu1.frbit.net/public/auth/login

Login/Register

Action

POST /auth/login

Request

  • email - User's email address.
  • username - User's username.
  • device_token - device token (push notification)

Response

{
    "id": "1",
    "email": "saakyz@gmail.com",
    "username": "n3tr",
    "token": "21345o34567899765thsdfgdsasrgfdcvtfbcv4",
    "created_at": "2013-08-23 07:22:36",
    "updated_at": "2013-08-23 07:22:36"
}

Events

Action

GET /event/create

Request

  • access_token - !important: User's email
  • where_text - String to define context of event.
  • location - location string.

Response

{
    "where_text": "test",
    "start_point": 5,
    "status": 1,
    "user_id": 3,
    "location": "",
    "updated_at": "2013-08-23 17:19:44",
    "created_at": "2013-08-23 17:19:44",
    "id": 1
}
